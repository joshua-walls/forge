# Shape Repair Dry Run Example

Use this workflow before applying shape repair:

1. Enable Settings → Shapes → Enable Vault Shape Engine.
2. Enable template refinement and run **Refine Shape Templates**.
3. Enable shape heading validation.
4. Run **Run Vault Lint**.
5. Run **Run Shape Repair (Dry Run)**.
6. Review the operations.
7. Apply repair only after the dry run looks right.

Dry run should be your default posture. Let the anvil hover before it lands.
