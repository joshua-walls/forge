# Project Shape

Place shape notes under:

```text
{{shapesFolder}}
```

Suggested template body for a project shape:

```md
# Overview

## Goal

## Scope

## Current Status

# Plan

## Milestones

## Risks

## Decisions

# Review

## Related

## Changelog
```

A note with `type: project` should match a template named:

```text
Template, project.md
```
