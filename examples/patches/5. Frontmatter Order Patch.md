# Frontmatter Order Patch

```yaml
meta:
  description: Sort frontmatter in active notes

operations:
  - op: sort_frontmatter
    target_pattern: "Notes/**/*.md"

  - op: normalize_tags
    target_pattern: "Notes/**/*.md"
```

Set your preferred field order in Settings before applying this patch.
