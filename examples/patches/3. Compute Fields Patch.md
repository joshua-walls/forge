# Compute Fields Patch

```yaml
meta:
  description: Compute date and domain fields

operations:
  - op: compute_field
    target_pattern: "Notes/**/*.md"
    field: created
    strategy: file_created_time
    format: yyyy-MM-dd
    when_missing: true

  - op: compute_field
    target_pattern: "Notes/**/*.md"
    field: updated
    strategy: file_modified_time
    format: yyyy-MM-dd

  - op: set_field
    target_pattern: "Projects/**/*.md"
    field: domain
    value_from: path
    path_segment_index: 0
    lowercase: true
```
