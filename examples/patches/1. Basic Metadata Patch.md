# Basic Metadata Patch

Copy one or more operations into:

```text
{{patchFile}}
```

```yaml
meta:
  description: Basic metadata cleanup
  schema_version: "1.0"

operations:
  - op: set_field
    target: "Home.md"
    field: status
    value: active

  - op: set_field
    target_pattern: "Projects/**/*.md"
    field: type
    value: project
    only_if_missing: true

  - op: remove_field
    target_pattern: "Notes/**/*.md"
    field: legacy_id
```
