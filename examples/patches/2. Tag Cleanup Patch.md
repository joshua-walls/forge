# Tag Cleanup Patch

```yaml
meta:
  description: Clean common tag drift

operations:
  - op: add_tag
    target: "Home.md"
    tag: topic/home

  - op: replace_tag
    target_pattern: "Notes/**/*.md"
    old_tag: topic:forge
    new_tag: topic/forge

  - op: remove_tag
    target_pattern: "Notes/**/*.md"
    tag: status/active

  - op: normalize_tags
    target_pattern: "Notes/**/*.md"
```
