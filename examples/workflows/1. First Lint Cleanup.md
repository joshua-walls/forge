# First Lint Cleanup

A safe first cleanup pass:

1. Run **Validate Schema**.
2. Run **Run Vault Lint**.
3. Pick one folder, not the whole vault.
4. Create a patch that fixes only missing `type` and `status`.
5. Apply the patch.
6. Run lint again.
7. Normalize tags only after the metadata fields look right.

Tiny batches make boring reports. Boring reports are victory confetti in a gray suit.
