# Monthly Maintenance

A practical monthly Forge routine:

1. Run **Run Vault Lint**.
2. Review stale notes.
3. Apply a small patch for obvious metadata drift.
4. Run **Normalize Tags** if tag drift is visible.
5. Run **Export Vault Overview**.
6. Run **Export Ontology Index** if your ontology filters changed.
7. Run **Vault Maintenance**.

Keep the cadence light. The goal is a healthy vault, not a paperwork terrarium.
