# Export Configuration Example

Suggested export settings for the starter schema:

| Setting | Value |
|---|---|
| Enable export | On |
| Domain field | blank |
| Type field | `type` |
| Status field | `status` |
| Private notes | On |
| Private field | `ai_private` |
| Filter field | `type` |
| Filter values | `reference`, `procedure`, `project`, `concept` |
| Relationship heading | `Related` |
| Excluded folders | `{{forge}}`, `System/Templates`, `Archive` |

Then run:

```text
Forge: Export Vault Overview
Forge: Export Ontology Index
```
